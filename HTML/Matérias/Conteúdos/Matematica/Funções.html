Funções

1. O que é uma função?

Por exemplo, uma função de A em B significa associar cada elemento pertencente ao conjunto A a um único elemento do conjunto B, sendo assim, um valor de A não pode estar ligado a dois valores de B.

Definição de Função

A notação para função é representada por: f: A → B (lê-se: "f de A em B").

Representação das Funções

Em uma função f: A → B, o conjunto A é chamado de domínio (D) e o conjunto B é denominado contradomínio (CD).

  • Um elemento de B relacionado a um elemento de A recebe o nome de imagem pela função.
  • Agrupando todas as imagens de B, obtemos o conjunto imagem, que é um subconjunto do contradomínio.

Exemplo:

Considere os conjuntos:

  • A = {1, 2, 3, 4}
  • B = {1, 2, 3, 4, 5, 6, 7, 8}

Se a função que determina a relação entre os elementos é dada por f(x) = 2x, então:

  • Para cada x do conjunto A, temos f(x) no conjunto B.
  • Assim, os valores de B relacionados aos elementos de A serão {2, 4, 6, 8}.

Tipos de Funções

As funções podem ser classificadas de acordo com suas propriedades. Confira os principais tipos:

Função Sobrejetora

Na função sobrejetora, o contradomínio é igual ao conjunto imagem. Portanto, todo elemento de B é imagem de pelo menos um elemento de A.

Notação: f: A → B, ocorre que Im(f) = B

Função Injetora

Na função injetora, todos os elementos de A possuem correspondentes distintos em B, e nenhum dos elementos de A compartilham a mesma imagem em B.

Função Bijetora

Na função bijetora, os conjuntos apresentam o mesmo número de elementos relacionados. Essa função é simultaneamente injetora e sobrejetora.

Função Inversa

A função inversa é um tipo de função bijetora, por isso é sobrejetora e injetora ao mesmo tempo.

Função Par

Uma função é par quando f(-x) = f(x), ou seja, a função possui a mesma imagem para x e para -x.

Função Ímpar

Uma função é ímpar quando f(-x) = -f(x). O gráfico de uma função ímpar é simétrico em relação à origem.

Função Composta

A função composta é um tipo de função matemática que combina duas ou mais funções. Pode ser representada como:

  • fog(x) = f(g(x))
  • gof(x) = g(f(x))

Função Modular

A função modular associa elementos em módulos e seus números são sempre positivos.

Função Afim

A função afim, também chamada de função do 1º grau, apresenta uma taxa de crescimento e um termo constante. Sua equação é dada por:

f(x) = ax + b

  • a: coeficiente angular
  • b: coeficiente linear

Função Linear

A função linear é um caso particular da função afim, onde a equação é dada por:

f(x) = ax

Quando o coeficiente a for igual a 1, a função linear é uma função identidade.

Função Quadrática

A função quadrática é chamada de função do 2º grau e é representada por:

f(x) = ax² + bx + c, sendo a ≠ 0.

  • a, b e c: coeficientes da função polinomial de grau 2.

Função Logarítmica

A função logarítmica de base a é representada por:

f(x) = loga(x), sendo a > 0 e a ≠ 1.

Ao invertermos a função logarítmica, obtemos uma função exponencial.

Função Exponencial

A função exponencial apresenta uma variável no expoente, com a base sendo maior que zero e diferente de 1:

f(x) = ax, sendo a > 0 e a ≠ 1.

Função Polinomial

A função polinomial é representada por expressões polinomiais:

f(x) = an * xn + an-1 * xn-1 + ... + a2 * x² + a1 * x + a0

  • an, an-1, ... , a2, a1, a0: números complexos
  • n: número inteiro
  • x: variável complexa

Funções Trigonométricas

As funções trigonométricas estão relacionadas com as voltas no ciclo trigonométrico:

  • Função Seno: f(x) = sen(x)
  • Função Cosseno: f(x) = cos(x)
  • Função Tangente: f(x) = tg(x)

Gráfico de uma Função

A maneira como um elemento y se relaciona com um elemento x é expressa através de um gráfico, que nos dá a ideia do comportamento da função. Cada ponto no gráfico é dado por um par ordenado de x e y, onde x é o valor de entrada e y é o resultado da relação definida pela função.

Imagine que você tenha uma "máquina", que recebe um número como entrada (x) e retorna outro número como saída (y). Se a máquina for uma função, para cada valor de x, haverá um único valor correspondente de y. Por exemplo, a função f(x) = x + 2, transforma qualquer valor de x somando 2 a ele, ou seja, f(3) = 5.

Em notação matemática, uma função é representada como f(x), onde "f" é o nome da função e "x" é o valor de entrada. O resultado da função é f(x) = y, ou seja, a função atribui a cada valor de "x" um único valor de "y". As funções podem ser representadas de várias formas, sendo as mais comuns as representações analíticas (equações), gráficas (curvas) e tabulares (tabelas de valores).

Exemplo: Para a função f(x) = x², se substituirmos x por 2, obtemos f(2) = 4. Assim, para cada número inserido, a função retorna o seu quadrado.

2. Elementos de uma função

As funções possuem três elementos fundamentais que determinam seu comportamento: o domínio, o contradomínio e a imagem. Entender esses elementos é essencial para resolver problemas de função em vestibulares, especialmente no ENEM, onde a interpretação de funções aparece com frequência.

  • Domínio: O domínio de uma função é o conjunto de todos os valores de entrada possíveis (valores de x). Esses valores são os números que podem ser substituídos na função.
  • Contradomínio: O contradomínio é o conjunto de todos os possíveis resultados da função (valores de y). Porém, nem todos os elementos do contradomínio são necessariamente atingidos pela função.
  • Imagem: A imagem é o conjunto dos valores de saída que são efetivamente atingidos pela função, ou seja, os resultados reais para os valores de entrada dados.

Por exemplo, se temos a função f(x) = x² com o domínio {–3, –2, –1, 0, 1, 2, 3}, o contradomínio será o conjunto dos quadrados desses números, ou seja, {0, 1, 4, 9}. A imagem, no entanto, será o conjunto dos resultados reais atingidos pela função, que é {0, 1, 4, 9}.

3. Tipos de funções

Existem vários tipos de funções, e cada tipo tem suas próprias características e aplicações. Para vestibulares como o ENEM, é importante conhecer os principais tipos de funções e como trabalhar com eles. Vamos explorar os dois tipos mais comuns: funções do 1º grau e funções do 2º grau.

Função do 1º Grau (Função Linear)

A função do 1º grau, também chamada de função linear, é uma função polinomial onde o grau da variável é 1. Ela é expressa por:

f(x) = ax + b

  • a: coeficiente angular, que indica a inclinação da reta.
  • b: coeficiente linear, que determina o ponto de interseção com o eixo y.

Exemplo de função do 1º grau:

f(x) = 2x + 3

Para esta função, quando x = 0, temos f(0) = 3, ou seja, a reta corta o eixo y no ponto (0, 3).

Gráfico de uma Função do 1º Grau

O gráfico de uma função linear é uma reta. A inclinação dessa reta é dada pelo coeficiente angular a, e o ponto de interseção com o eixo y é dado por b.

Função do 2º Grau (Função Quadrática)

A função do 2º grau, também conhecida como função quadrática, é uma função polinomial em que o maior grau da variável é 2. Sua expressão é dada por:

f(x) = ax² + bx + c, onde:

  • a, b e c são coeficientes reais e a ≠ 0.

Exemplo de função do 2º grau:

f(x) = x² - 4x + 3

Essa função é uma parábola. O gráfico de uma função do 2º grau é sempre uma parábola, podendo ser voltada para cima ou para baixo, dependendo do valor de a.

Raízes da Função Quadrática

As raízes de uma função quadrática são os valores de x para os quais f(x) = 0. Para encontrar as raízes, usa-se a fórmula de Bhaskara:

x = (-b ± √(b² - 4ac)) / 2a

Exemplo:

  • Para a função f(x) = x² - 4x + 3, temos a = 1, b = -4 e c = 3.
  • Substituindo na fórmula de Bhaskara, obtemos:

    • x = (-(-4) ± √((-4)² - 4 * 1 * 3)) / (2 * 1)

    x = (4 ± √(16 - 12)) / 2

    x = (4 ± √4) / 2

    x = (4 ± 2) / 2

    • x₁ = (4 + 2) / 2 = 6 / 2 = 3
    • x₂ = (4 - 2) / 2 = 2 / 2 = 1

    Logo, as raízes da função são x₁ = 3 e x₂ = 1.

Gráfico de uma Função Quadrática

O gráfico de uma função quadrática é uma parábola. A direção da parábola depende do valor de a:

  • Se a > 0, a parábola abre para cima.
  • Se a < 0, a parábola abre para baixo.

O vértice da parábola é o ponto onde a função atinge seu valor máximo ou mínimo, e a equação da reta de simetria é dada por:

x = -b / 2a

Resumo

As funções do 1º e 2º grau são fundamentais para entender o comportamento das equações lineares e quadráticas, e seus gráficos desempenham um papel importante na visualização dessas relações.

5. Como as funções são cobradas no ENEM

No ENEM, as questões de Matemática que envolvem funções costumam ser interpretativas. O candidato precisa não apenas entender a teoria das funções, mas também saber como aplicá-las em situações do cotidiano, como vimos nos exemplos anteriores.

É importante que você entenda como identificar a forma de uma função a partir de um gráfico ou uma situação problema e consiga interpretá-la corretamente. Questões como identificar as raízes de uma equação quadrática ou calcular o valor de uma função em um determinado ponto são comuns.

Além disso, o ENEM também gosta de explorar a interpretação de gráficos de funções, pedindo para o candidato analisar a variação de uma variável em função de outra, como em gráficos de crescimento populacional ou de variação de temperatura ao longo do tempo.

6. Conclusão

Estudar funções matemáticas é essencial para o sucesso no ENEM e em outros vestibulares. Saber interpretar, resolver e aplicar funções é uma habilidade que será exigida em várias questões de Matemática. Além disso, entender o papel das funções no cotidiano e nas diferentes áreas do conhecimento é fundamental para a compreensão de fenômenos naturais e sociais.
